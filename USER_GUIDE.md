## Game Overview

Complete the Word is a strategic multiplayer word game where players collaboratively build words by adding one letter at a time. The goal is to be the last player remaining by avoiding moves that make it impossible to form valid words.

## Getting Started

1. Open the game in your web browser
2. Choose your preferred theme
3. Select an input method (Keyboard or Microphone)
4. Set the number of players and enter their names
5. Click "Start Game"

## Game Rules

1. Players take turns adding one letter to the current word fragment
2. Each added letter must keep the word fragment as a valid prefix for at least one English word
3. Players can continue adding letters even after forming a complete word
4. A player is eliminated if their letter makes it impossible to form any valid word
5. The last player remaining wins the game
6. Each player has 10 seconds to make their move

## How to Play

### Starting the Game
1. On the start screen, select from 6 beautiful themes
2. Choose your input method:
   - **Keyboard**: Type letters manually
   - **Microphone**: Speak letters aloud (auto-submits)
3. Set the number of players (2-8)
4. Enter player names
5. Click "Start Game"

### During Gameplay
1. The current player's name is highlighted with a golden border
2. The timer counts down from 10 seconds
3. Enter a single letter using your chosen input method:
   - **Keyboard**: Type a letter and press Enter or click Submit
   - **Microphone**: Clearly say a letter (auto-submits)
4. Valid letters are added to the word fragment
5. Complete words are displayed in the "Completed Words" section
6. Play continues until only one player remains

## Input Methods

### Keyboard Input
- Type any letter from A-Z in the input field
- Press Enter or click the Submit button to add the letter
- The input field clears automatically for the next player

### Microphone Input
- Click "Allow" when prompted to enable microphone access
- Speak a single letter clearly (e.g., "A", "B", "C")
- Letters are automatically submitted after recognition
- The microphone automatically restarts for continuous input
- If recognition fails, you can try again or switch to keyboard

## Game Interface

### Players Panel
- Shows active players with timers
- Current player is highlighted with a golden glow
- Eliminated players appear in the "Eliminated" section

### Game Board
- Displays the current word fragment
- Shows game status messages
- Contains the letter input section

### Timer
- Counts down from 10 seconds per turn
- Changes color as time runs low (green → yellow → red)
- Player is eliminated if time expires

### Controls
- **Mic Button**: Toggle microphone on/off (disabled during game)
- **Restart Button**: Restart the entire game

### Completed Words
- Shows all valid words formed during gameplay
- Updates in real-time as words are completed

## Troubleshooting

### Microphone Not Working
1. Ensure your browser supports speech recognition (Chrome or Edge recommended)
2. Check that you've granted microphone permissions
3. Speak clearly and at a moderate pace
4. Ensure you're in a quiet environment

### Game Not Loading
1. Refresh the page
2. Check your internet connection
3. Ensure JavaScript is enabled in your browser

### Letter Not Accepted
1. Make sure you're entering only single letters (A-Z)
2. Verify that the letter creates a valid word prefix
3. Check that you're not exceeding the time limit

### Mobile Issues
1. Ensure you're using a modern mobile browser
2. Check that pop-ups aren't blocked
3. Try rotating your device if elements appear cut off

## Tips for Success

1. Think ahead about possible word continuations
2. Try to create word fragments with multiple possible endings
3. Pay attention to the timer - don't rush, but don't dawdle
4. Use the completed words list for inspiration
5. When using microphone, speak clearly and at a consistent volume

Enjoy the game and may the best word-builder win!
